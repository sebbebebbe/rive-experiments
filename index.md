# Rive Experiments

Welcome to my collection of Rive experiments and examples!

## About

This repository contains various experiments and demonstrations using [Rive](https://rive.app/), a powerful tool for creating interactive animations.

## Projects

### Bindings
Examples demonstrating Rive bindings and integrations using WebGL2 renderer.

## Getting Started

Clone this repository and explore the different examples in each folder.

```bash
git clone https://github.com/sebbebebbe/rive-experiments.git
cd rive-experiments
```
